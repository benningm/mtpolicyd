# NAME

Mail::MtPolicyd - a modular policy daemon for postfix

# VERSION

version 2.04

# DESCRIPTION

Mail::MtPolicyd is the Net::Server class of the mtpolicyd daemon.

## SYNOPSIS

    use Mail::MtPolicyd;
    Mail::MtPolicyd->run;
